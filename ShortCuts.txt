10 keyboard Combinations that can make your life yeasy
1. ctrl + z - undo
2. ctrl + n + shift - reate a new folder
3. shift + delete - deletes a file permanentely.
4. windows + . - shows emojy.			
5. windows + m - minimize all windows 
6. windows + l - locks the windows
7. windows + ctrl + d - creaza un desktop virtual
8. windows + ctrl + -> - face mutarea printre descktopurile virtuale
9. windows + ctrl + f4 - inchide descktopul virtual.
10. ctrl + n - deschide o fereastra noua in chrome.
11. ctrl + t - deschide un nou tab in chrome existent.		
12. alt + tab - face switch intre programele windows. sau alt + segetile
13. ctrl + tab - navigeaza printre tab-urile chrome
14. ctrl + un numar te duce la tab-ul cu numarul respectiv din chrome
15. ctrl + 9 - te duce la ultimul tab din chrome.
16. ctrl + shift + n - incognito mode in chrome.
17. ctrl + shift + t - redeschide ultimul tab din chrome care a fost inchis. putem sa repetam de 10 ori si redeschidem ultimile 10 inchise taburi
18. ctrl + click - pe un link deschide acest link intr-o fereastra noua
19. recuva - programa care recupereaza fisierele sterse permanent de pe calculator.
20. ctrl + q - inchide browser-ul.
21. ctrl + alt + sageti = roteste screen-ul.
22. windows + d - arata desctopul iar daca mai repetam o data pune tot la loc.
23. windows + u - shut the computer down
24. windows + r - restarts the computer
25. windows + h - will hibernate the computer
26. windows + s - will sleep the computer.
27. windows + tab - arata cate descktopuri ai deschise si cate ferestre.
28. ctrl + tab - schimba taburile din visual studio
29. npx create-react-app the-name - se creaza un nou proiect react
30. npm install react-router-dom - se instaleaza react router
