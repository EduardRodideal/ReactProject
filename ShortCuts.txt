node -v - comanda care introdusă în command promt afișează versiunea Node.js. (npm)

Window key + m - minimizează toate ferestrele deschise (le pune jos pe bară).
npx create-react-app my-app; cd my-app; npm start - prin aceste comenzi se crează o React App.
Ctrl + shift + j = deschide consola in browser.
